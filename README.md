# Slate
Modular multithreaded game engine

## Compiling

### Requirements
- Odin Compiler
### Windows
```cmd
  ./build.bat
```
### Linux
```cmd
  ./build.sh
```
## Running
`./slate.bin` on **Linux** or `./slate.exe` on **Windows**

The slate executable will search for a `config.txt` file and a `mods/` directory wherever it's executed
